# 2.ElasticSearch单节点的部署

（1）从指定仓库下载ElasticSearch镜像
```
    docker pull docker.elastic.co/elasticsearch/elasticsearch:6.2.4
```


（2）建立容器局域网

```
   docker network create mynetwork
```

（3）检测局域网

```
   docker network inspect mynetwork
```

（4）加载镜像生成Elastic的Docker容器

```
    docker run -p 9200:9200 -p 9300:9300 -e "discovery.type=single-node" -e ES_JAVA_OPTS="-Xms1g -Xmx1g" --name elastic --network mynetwork -d docker.elastic.co/elasticsearch/elasticsearch:6.2.4
```

（5）测试ElasticSearch是否可用

```
   curl http://localhost:9200
```
