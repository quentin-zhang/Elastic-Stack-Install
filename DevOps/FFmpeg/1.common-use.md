# 1.ffmpeg常用方法

1. 查看视频信息

```bash
ffmpeg -i a.mp4 -hide_banner
```

2. 更改视频文件分辨率

```bash
ffmpeg -i a.mp4 -s 640x480 -c:a copy b.mp4
```

3. 每秒抽取一帧

```bash
ffmpeg -i b.mp4 -r 1 -f image2 image-%05d.jpeg
```

4. 取关键帧

```bash
ffmpeg -i b.mp4 -vf select='eq(pict_type\,I)' -vsync 2 -s 640*480 -f image2 keyframe-%02d.jpeg
```
